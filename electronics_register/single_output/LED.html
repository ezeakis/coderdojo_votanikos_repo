Συνδεσμολογία

Θετικό ποδαράκι του LED –> GPIO 4

Αρνητικό ποδαράκι του LED –> Ground

Interactive mode

from gpiozero import LED

myled = LED(4)

myled.on()

myled.off()

Script mode

from gpiozero import LED

from time import sleep

myled = LED(4)

while True:

     myled.on()

     sleep(1)

     myled.off()

     sleep(1)

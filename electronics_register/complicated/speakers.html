Οδηγίες

Σε αυτή την κατασκευή δεν έχουμε python εντολές.

Δίνουμε τις εντολές στο περιβάλλον που βρισκόμαστε μόλις μπούμε στο Raspberry Pi.

Χωρίς να πατήσουμε python ή nano.

Αυτό το είδος εντολών το ονομάζουμε bash ή shell

Επίσης, σε αυτή την κατασκευή, αν θέλετε μπορείτε να δώσετε τα δικά σας mp3 αρχεία σε έναν mentor για να μεταφερθούν στο raspberry pi και να είναι διαθέσιμα.



Υλικά

1 Raspberry Pi

1 τροφοδοτικό

1 Ηχεία



Συνδεσμολογία

Το καλώδιο των ηχείων στην υποδοχή ήχου του Raspberry Pi



bash εντολές

ls

Θα μας βγάλει μία λίστα με τα αρχεία που έχει ο φάκελος που είμαστε.

Αν, για παράδειγμα, έχουμε ένα mp3 αρχείο με το όνομα a.mp3, τότε δίνουμε

mpg321 a.mp3

Για να το διακόψουμε πατάμε Ctrl+C

Αν θέλουμε να αλλάξουμε την ένταση του ήχου (πχ στο 80%) γράφουμε

amixer cset numid=1 80%



Βοηθητικές εντολές

Για να δούμε σε ποιον φάκελο βρισκόμαστε δίνουμε

pwd

Για να πάμε στον φάκελο δίνουμε/a/b

cd /a/b

Αν μας εμφανίσει μήνυμα ότι δεν ξέρει τι είναι το mpg321, αυτό σημαίνει ότι το πρόγραμμα δεν υπάρχει.

Μπορούμε να το κατεβάσουμε με αυτή την εντολή.

sudo apt-get install mpg321
